# Azure Marketplace Purchase Prerequisites

## Overview
This guide outlines the prerequisites for purchasing Entelexos through the Azure Marketplace.

## Required Azure Resources
- Active Azure subscription
- Appropriate permissions in your Azure tenant
- Sufficient quota for the required resources
- Network configuration requirements

## Account Requirements
- Azure account with billing enabled
- Appropriate role assignments
- Access to Azure Marketplace
- Ability to create and manage resources

## Network Requirements
- Virtual Network configuration
- Network Security Groups
- Required ports and protocols
- DNS configuration

## Cost Considerations
- Subscription type requirements
- Resource cost estimates
- Billing setup
- Cost management tools
