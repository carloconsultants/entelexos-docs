# Power BI Prerequisites

## Overview
This guide outlines the prerequisites for using Entelexos with Power BI.

## Power BI Requirements
- Power BI Pro or Premium license
- Appropriate permissions in Power BI service
- Access to required workspaces
- Data gateway configuration (if needed)

## Data Source Requirements
- Supported data source types
- Connection requirements
- Authentication methods
- Data refresh settings

## Workspace Configuration
- Workspace creation and setup
- Role assignments
- Capacity requirements
- Sharing and collaboration settings

## Security Requirements
- Row-level security setup
- Data privacy settings
- Authentication methods
- Access control configuration
